# Codescribe v0.12.2 Release Notes Draft

Codescribe `v0.12.2` is the public-readiness release for the current native macOS dictation and assistive voice stack.

## Highlights

- Local Whisper Live preview remains the primary low-latency path for dictation.
- Assistive voice-chat rendering now avoids the first-message double-render path.
- Drawer card disk I/O moved off the main thread to reduce AppKit stalls.
- Onboarding now refreshes when permission state drifts and avoids always-on-top focus behavior.
- Raw recording stops use the correct final-pass truth path.
- Dictation cleanup preserves codescribe/Vibecrafted vocabulary such as Loctree.
- Public release metadata, changelog, release checklist, and governance files are aligned for launch.

## Install

Source install is the guaranteed path until the `v0.12.2` DMG is published from GitHub Actions and passes Gatekeeper smoke.

```bash
git clone https://github.com/vetcoders/codescribe.git
cd codescribe
make install-app
```

## DMG Release Gate

Do not announce a release DMG until all of these are true:

- The `Release DMG` workflow runs on tag `v0.12.2`.
- The workflow uses Developer ID signing and Apple notarization.
- `Codescribe_0.12.2.dmg` is uploaded to the GitHub Release.
- The downloaded DMG installs and launches on a non-development Mac without Gatekeeper workarounds.

## Known Public-Launch Blockers

- The latest live GitHub Release observed before this draft was `v0.8.0`.
- GitHub repository visibility and launch metadata were corrected on 2026-06-23.
- The hardened signed/notarized release workflow must land on `main` before creating the `v0.12.2` tag; the old `main` workflow still builds an ad-hoc DMG.
- GitHub Actions signing/notary secrets were not listed on 2026-06-23.
- GitHub Pages still served the 2026-05-07 landing before this branch was deployed.
- A local `Codescribe_0.12.2.dmg` observed on 2026-06-23 was rejected by `spctl` with `source=Insufficient Context`; it is not acceptable release evidence.
