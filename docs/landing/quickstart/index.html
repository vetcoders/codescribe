CodeScribe quickstart for real first use.

This page is the public install path. No hidden markdown, no broken internal repo links. Install, grant the required macOS permissions, pick your transcription backend, and get to the first solid transcript.

Install from source Open README quick start

1. Install

Release DMG

Use a GitHub Release only after current v0.12.x standard and full DMGs exist and are signed, notarized, and smoke-tested. Until then, source install is the honest public path.

Build from source

The supported local install path is:

git clone https://github.com/VetCoders/CodeScribe.git
cd CodeScribe
make install-app

2. Grant permissions

  • Microphone
  • Accessibility
  • Input Monitoring

Without these, global hotkeys and live dictation preview will not be truthful.

3. Pick the runtime shape

Transcription overlay ON

Use this for low-latency live preview. Partial transcript updates are shown during capture and tuned from Settings → Transcription.

Transcription overlay OFF

Use this for a quieter, more buffered runtime profile when live overlay is not needed and you want lower local load.

Backend selection

Settings → Transcription lets you choose between local Whisper and Cloud STT. Cloud in the current build is a post-capture/final path, not live cloud overlay streaming.

4. First transcript

  • Open Settings from the menu bar icon.
  • Confirm your preferred backend in Transcription.
  • Check overlay state in Audio & Input.
  • Use the default Dictation hotkey: hold Fn/Globe.
  • Speak, release, and verify the transcript lands in the frontmost app.
